# 学びメモ

## Q. パーサジェネレータとは？
パーサジェネレータは、**文法（構文規則）からパーサ（構文解析器）を自動生成するツール**です。手書きで再帰下降パーサを実装する代わりに、BNF/EBNF などで文法を書けば解析コードが生成されます。

### ざっくり流れ
1. 文法を書く
2. 生成ツールを実行
3. パーサコードが出力される
4. AST 構築やエラーハンドリングに集中できる

### メリット
- 文法が仕様として明文化される
- 複雑な文法でも保守しやすい
- 変更に強い（文法修正 → 自動生成）

### 注意点
- 文法の書き方に慣れが必要
- 生成コードがブラックボックスに見えることがある
- 小さい言語なら手書きパーサの方が簡単な場合もある

### 代表例
- Yacc/Bison（LALR, C系）
- ANTLR（LL系, 多言語）
- Menhir（OCaml）
- PEG.js / pest / nom（PEG系）
